13.01.2017

New search feature!

It's now possible to search specifically for mana (and other) symbols in the rules text of cards.

To do that, you write "{" + symbol key + "}".

For example, the key for the tap symbol is "tap", so writing {tap}" searches for all cards that have an activated tap ability.

Some other symbol keys are:

{W}{U}{B}{R}{G}
(regular mana) 
{0}{8}{X}
(generic mana) 
{C}
(colorless mana) 
{WP}{UP}{BP}{RP}{GP}
(phyrexian mana) 
{tap}{untap}
(tap + untap action)

Also, the Import page now supports a different kind of card identification, "set abbreviation + # + card number".

For instance,

4 bfz#123

imports 4 copies of card number 123 from the set Battle for Zendikar.

(This is for when you don't know the English name of the card you want to import. Hope it helps!)
